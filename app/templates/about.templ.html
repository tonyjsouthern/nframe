What is N-Frame?

N-Frame is an application the helps mobile developers and enthusiasts create professional looking displays for their application


How can I get new phones?

Currently phones are only available to be added by me. If you would like a new phone to be supported you can email me at tonyjsouthern@gmail.com and I will work on adding it as quickly as possible


What features are going to be implemented?

As this website was developed by a one man team fetaures may be slow to roll out. the following features are currently in plans for support:

Hex/RGB/HSL background color selector Font style selector
Toggle background blur
Adjustable drop shadow on phone frames
Room for a second logo/other image


I've found a bug, where can I submit a report?

Bug reports can be submitted to me directly at tonyjsouthern@gmail.com
